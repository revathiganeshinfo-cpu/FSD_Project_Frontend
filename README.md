🚀 Features
👤 User Features
🔍 Search restaurants by name, location, cuisine, price
🍴 View restaurant details
📅 Book table reservations
💳 Online payment integration (Stripe)
📖 View booking history
🛠️ Admin Features
📊 Dashboard overview (Users, Restaurants, Reservations, Revenue)
🍽️ Manage Restaurants (Add/Edit/Delete)
📅 Manage Reservations (CRUD)
💰 Track total revenue

Frontend
React.js
Tailwind CSS
Framer Motion
Axios
Backend
Node.js
Express.js
Database
MongoDB (Mongoose)
Payment
Stripe API
